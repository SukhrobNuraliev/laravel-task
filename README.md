## Laravel Task

### Setup
- ```cp .env.example .env```
- ```composer install```
- ```npm install```
- ```php artisan key:generate```
- ```php artisan storage:link```
- ```php artisan sail:install```
- ```./vendor/bin/sail up```
- ```npm run dev```
- ```atisan queue:work```
- ```./vendor/bin/sail artisan migrate --seed```
